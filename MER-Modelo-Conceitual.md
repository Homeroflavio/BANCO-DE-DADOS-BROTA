# O modelo conceitual representa todas as entidades do sistema:
- Espécie
- Cultivar
- Fornecedor
- Armazém
- Endereço
- Município
- Usuário
- Agricultor
- Lote
- Movimentação
- OrdemExpedição
- ItemOrdemExpedição
- Entrega

Cada relacionamento foi definido com cardinalidades, dependências e opcionalidades.
