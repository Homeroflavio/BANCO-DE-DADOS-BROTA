# Sistema de Gestão de Sementes – Projeto Integrador (PI) – Banco de Dados

## Minimundo do Projeto

O sistema de Gestão de Sementes tem como objetivo controlar todo o fluxo de sementes,
desde a entrada dos lotes até a entrega final aos agricultores. O governo ou instituição 
responsável precisa acompanhar espécies, cultivares, fornecedores, armazéns, 
movimentações, ordens de expedição e entregas.

O sistema permite:

- Cadastro de espécies e cultivares.
- Registro de fornecedores e agricultores.
- Controle de lotes, incluindo peso, laudos e validade.
- Armazenamento físico dos lotes em armazéns.
- Registro de movimentações (entrada, saída, transferência).
- Geração de ordens de expedição e seus itens.
- Controle das entregas feitas aos agricultores.
- Rastreamento completo da origem e destino de cada lote.

Todo o banco foi modelado, normalizado e implementado em MySQL.
